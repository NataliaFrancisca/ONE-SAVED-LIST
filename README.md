# ONE. SAVED LIST

A website where you will be able to save everthing you need to remember.

`On this moment I am building this project!`

### Functionality:
    1. Register and Login
    2. Save a content
    3. Search for a content 
    4. Apply filter
    5. View your content
    6. Update your data
    7. Choose a user photo
    8. Logout
    9. Delete account
